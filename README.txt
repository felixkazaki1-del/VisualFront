
OKSystem Advanced

Features
- Import system from PluralKit
- Add / remove alters from front
- Front history timeline
- Search bar
- Dark / light theme
- Markdown bold (**text**)
- Alter color tint
- Export / import backups
- Responsive layout

Usage
1. Open index.html
2. Enter your PluralKit system ID
3. Import members
4. Front alters and track history
